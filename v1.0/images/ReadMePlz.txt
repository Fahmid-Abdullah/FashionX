How to:
1. Go to the following site and download the dataset: https://www.kaggle.com/datasets/paramaggarwal/fashion-product-images-dataset?resource=download

2. Extract the files and copy the images from the extracted zip to the images directory in this repo (current)

3. Run create_csv.py

4. Setup is complete and now you can work with main.py, model.py, testing.py
